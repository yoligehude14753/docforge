### Added
- Initial open-source release
- Multi-provider AI support: OpenAI, DeepSeek, Claude, Ollama
- Document pipeline: requirements analysis → outline generation → per-section generation → refinement
- RAG-enhanced writing with PDF and Word document parsing
- Local vector store with TF-IDF embeddings
- Word (.docx) export
- Document types: bid response, solution document, project description, business proposal, service plan, custom
- 18-module Vitest test suite
- Tauri 2 desktop shell (macOS, Windows, Linux)
